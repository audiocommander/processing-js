Name

Acceleration
Examples 
		
void draw() {

  fill(255);
  // map acceleration values from the expected acceleration.x range (-10 to +10) to the full width (0 to width)
  int x = map(acceleration.x, -10, 10, 0, width);
  rect(x, 100, 100, 100);
  
}
Description The Acceleration object contains the acceleration data for the device in m/s^2 along the x, y and z axes.
In contrast to the orientation object, it is available on most devices.
 
For further informations see html5rocks.com/en/tutorials/device/orientation/ and dev.w3.org/geo/api/spec-source.html.
 
Note that this object is only available in Processing-mobile.js and not compatible with Processing.
Fields (float) x: device acceleration in m/s^2 along the X axis
(float) y: device acceleration in m/s^2 along the Y axis
(float) z: device acceleration in m/s^2 along the Z axis
Syntax acceleration.x
acceleration.y
acceleration.z
