Name

Touch
Examples 
		
void touchMove(TouchEvent t) {

  if(t.touches.length > 0) {

    Touch aTouch = t.touches[0];

    println("Touch ID " + aTouch.identifier + " moved to " + aTouch.offsetX + "," + aTouch.offsetY);

  }
  
}
Description The Touch object contains various information about a touch, such as a universal identifier and various positions related to the coordinate system.
 
This is a preliminary document. For further informations see w3.org/TR/touch-events.
 
Note that this object is only available in Processing-mobile.js and not compatible with Processing.
