Description The Orientation object contains the tilt information for the device in degrees along the x, y and z axes.
In contrast to the acceleration object, it is only available on newer devices with a gyroscope/compass sensor. Additionaly the Orientation object also contains compass informations like heading and accuracy. If no such sensor is available in the device, the object returns either 0 or -1 values.
 
Note: While the Orientation object contains accurate realtime tilting data, a simple device orientation can also be easily determined by the JavaScript window.orientation value, which returns four int values: 0 (up), 90 (left), -90 (right) or 180 (down).
 
For further informations see html5rocks.com/en/tutorials/device/orientation/ and dev.w3.org/geo/api/spec-source.html.
 
Note that this object is only available in Processing-mobile.js and not compatible with Processing. There also may be incompatibilites if this sketch is run in older or non-webkit browsers.
Fields (float) alpha: the direction the device is facing according to the compass (float) beta: the angle in degrees the device is tilted front-to-back (float) gamma: the angle in degrees the device is tilted left-to-right (float) compassHeading: the iOS 5 compass heading in degrees (or -1) (float) compassAccuracy: the iOS 5 compass accuracy (or -1)
