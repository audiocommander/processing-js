Name

TouchEvent
Examples 
		
void touchMove(TouchEvent t) {

  println("All touches: " + t.touches.length);
  
  println("Target touches: " + t.targetTouches.length);

  println("Changed touches: " + t.changedTouches.length);

}
Description The TouchEvent contains three TouchLists: touches, targetTouches and changedTouches
 
This is a preliminary document. For further informations see w3.org/TR/touch-events.
 
Note that this object is only available in Processing-mobile.js and not compatible with Processing.
