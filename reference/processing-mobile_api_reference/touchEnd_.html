Name

touchEnd()
Examples 
void touchEnd(TouchEvent t) {

  println("touchEnd:" + t.changedTouches.length);
  
  int i;

  for (i=0; i<t.changedTouches.length; i++) {
	
    long id = t.touches[i].identifier;

  }

}
Description The touchEnd() function is called every time a known touch event has left the screen. Note that at this point the touch isn't available in the touches TouchList, as it is now gone. You will find all touches that ended in the changedTouches TouchList
 
This is a preliminary document. For further informations see w3.org/TR/touch-events.
 
Note that this object is only available in Processing-mobile.js and not compatible with Processing.
